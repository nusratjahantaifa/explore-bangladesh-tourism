# 🌍 Tourism Management Website

## 📌 Project Overview
This is a Tourism Management web application where users can explore tourist spots from South Asia, add their own spots, and manage their list.

---

## 🚀 Features
- 🔍 View all tourist spots
- ➕ Add new tourist spots (Private Route)
- 📋 Manage personal list (Private Route)
- 🔐 Authentication (Login/Register)
- 🎯 Responsive design

---

## 🛠️ Technologies Used
- React.js
- Tailwind CSS + DaisyUI
- Firebase Authentication
- MongoDB
- Express.js
